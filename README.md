## Dependencies

* Ubuntu 16.04 or newer.
* GCC 7.4 or newer.
* Miniconda3 x64 (latest version) (`conda env create --file environment.yml`).

## Building

* In the root project directory: `make` to build the program, `make test` to build the tests, and `make clean` to...clean.
